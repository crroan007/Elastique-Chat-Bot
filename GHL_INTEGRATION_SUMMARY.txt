HOW TO USE - 3 STEPS
================================================================================

STEP 1: IMPORT TO GHL KNOWLEDGE BASE
────────────────────────────────────
1. Open: ghl_knowledge_base.json in a text editor
2. Select all content (Ctrl+A)
3. Copy (Ctrl+C)
4. Go to: GoHighLevel Dashboard
   Path: Automation > Chatbots > [Your Bot] > Knowledge Base
5. Click: "Add Source" → "FAQs"
6. Paste content into import dialog
7. Click: "Save" or "Import"

Result: Your bot now knows all 30 Elastique products


STEP 2: CREATE PRODUCT RESPONSE TEMPLATE
─────────────────────────────────────────
1. Open: ghl_product_tiles.json in text editor
2. Find product you want to recommend
3. Copy the "content" field (entire HTML string)
4. Go to: GHL Bot > Response Templates
5. Create new template or edit existing response
6. Paste HTML tile wherever you want product to appear
7. Save template

Result: When bot recommends products, they display as nice tiles


STEP 3: TEST & DEPLOY
──────────────────────
1. Test your bot with product questions:
   - "Tell me about L'Original Leggings"
   - "What compression wear do you have?"
   - "Show me your best sellers"

2. Verify:
   - Bot pulls product info from Knowledge Base
   - Product tiles render correctly
   - Links go to product pages
   - All details are accurate

3. Deploy to production
4. Monitor conversations for engagement
5. Track revenue attribution (5% share per SOW)

================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

Issue: "Product tiles not showing in GHL chat"
Solution:
  1. Verify GHL plan supports HTML rich text
  2. Test HTML in GHL's rich text editor preview
  3. If HTML not supported, use text-only format from GHL_QUICK_START.md

Issue: "Product images not loading"
Solution:
  1. Check HTTPS URLs (all Shopify CDN URLs are HTTPS)
  2. Test image URL directly in browser
  3. Verify image still exists on Shopify
  4. Refresh catalog: python ghl_catalog_parser.py

Issue: "Sizes or colors missing"
Solution:
  1. Verify variants are set up in Shopify Product Admin
  2. Re-run: python ghl_catalog_parser.py
  3. Manually add to ghl_knowledge_base.json if needed
  4. Re-import to GHL

Issue: "Bot not recognizing product questions"
Solution:
  1. Verify KB entries are published in GHL
  2. Test exact phrases from Knowledge Base
  3. Check bot training/intent matching settings
  4. Add sample training conversations
