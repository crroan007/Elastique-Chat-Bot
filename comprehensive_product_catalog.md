# ELASTIQUE COMPREHENSIVE PRODUCT CATALOG & RECOMMENDATION LOGIC

## 1. RECOMMENDATION LOGIC (THE "BRAIN")

### BY SYMPTOM / GOAL
*   **"I have heavy legs / swelling / edema":**
    *   **Primary Rec:** **L'Original Leggings** (Maximum graduated compression).
    *   **Alternative:** **L'Original Stirrup Leggings** (If they want foot coverage/style).
    *   **Warm Weather:** **L'Original Bike Short** or **Flowy Short**.
*   **"I have cellulite / want better skin texture":**
    *   **Primary Rec:** **MicroPerle® Collections** (L'Original, Iconic, Fierce, Le Monde).
    *   **Why:** The beads massage the skin to stimulate superficial lymphatics.
*   **"I am traveling / flying":**
    *   **Primary Rec:** **L'Original Leggings** (Must have graduated compression for flight).
    *   **Alternative:** **Lisse Leggings** (Comfortable for long haul, but L'Original is better for swelling).
*   **"I want tummy control / core support":**
    *   **Primary Rec:** **Adorn Bodysuit** (Full core compression).
    *   **Alternative:** **Lisse Leggings** (High waist support).
*   **"I work out / sweat a lot":**
    *   **Primary Rec:** **Iconic Collection** (Performance fabric, sweat-wicking).
    *   **Alternative:** **Fierce Collection** (High intensity support).
*   **"I want something for everyday / work from home":**
    *   **Primary Rec:** **Lisse Collection** (Softer, "second skin" feel, all-day wear).

### BY TECHNOLOGY
*   **MicroPerle® (Beads):** L'Original, Iconic, Fierce, Le Monde, Adorn, Riviera.
    *   *Use for:* Cellulite, texture, sensory feedback, massage.
*   **Pure Compression (No Beads):** Lisse Collection.
    *   *Use for:* Swelling reduction, slimming, everyday comfort, sensitive skin.
