## Formatting Rules

1. Use horizontal rules (---) to separate major sections
2. Use **bold headers** for all section titles
3. Use [Text](URL) format for ALL hyperlinks
4. Add blank lines between each bullet point
5. Group sources by category (Studies, Videos, Products)
6. Always end with follow-up questions
